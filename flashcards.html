JavaScript

A programming language that is understood by browsers.

Operators

Constructs which behave generally like functions, but which differ syntactically or semantically from usual functions. Common simple examples include arithmetic and logical operations.

Variables

A value that can change, depending on conditions or on information passed to the program.

Variable Naming Conventions

The variable name has to be as descriptive as possible, while remaining concise and consistent. They should not include negative logic.

Functions

A function is a type of procedure or routine. Some programming languages make a distinction between a function, which returns a value, and a procedure, which performs some operation but does not return a value.

Methods

A procedure associated with an object. An object is made up of data and behavior, which form the interface that an object presents to the outside world. Data is represented as properties of the object and behavior as methods.
